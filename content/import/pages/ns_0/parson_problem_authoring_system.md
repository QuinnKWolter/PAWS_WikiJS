Parson Problem can be created using a tool called [Js-parsons Problem
Generator](http://acos.cs.hut.fi/jsparsons-generator) from [ACOS Content
Server](http://acos.cs.hut.fi/).

It Generates ACOS compatible
js-parsons json-formatted problem descriptions from given input. The
resulting page will give a JSON-reprsesentation of the exercise that can
be hosted anywhere and linked to JSON. Additionally, a direct URL will
be given that can be used to test and access the exercise.
