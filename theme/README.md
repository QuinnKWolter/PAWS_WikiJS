# PAWS Wiki Theme

This directory contains custom theme files for the University of Pittsburgh PAWS Wiki.

## Files

- `custom.css` - Main custom stylesheet with Pitt branding
- `logo.png` - University of Pittsburgh logo (to be added)
- `favicon.ico` - Site favicon (to be added)

## Installation

1. Copy the contents of `custom.css` into WikiJS:
   - Go to **Administration → Rendering → Custom CSS**
   - Paste the CSS content
   - Save

2. Upload logo:
   - Go to **Administration → UI → Logo**
   - Upload the official Pitt logo
   - Recommended size: 200px width, transparent background

3. Upload favicon:
   - Go to **Administration → UI → Favicon**
   - Upload `favicon.ico`

## Brand Colors

- **Pitt Royal Blue**: #003594
- **Pitt Gold**: #FFB81C
- **New Sky**: #DBEEFF
- **Medium Blue**: #00205B

## Customization

To customize the theme further, edit `custom.css` and update the CSS variables at the top of the file.

## Logo Requirements

The logo should be the official University of Pittsburgh "Institutional Mark" (shield + signature). 
- Use full color version when possible
- Maintain clear space around logo (height of the 'P' in "Pittsburgh")
- Do not skew, outline, or apply effects
