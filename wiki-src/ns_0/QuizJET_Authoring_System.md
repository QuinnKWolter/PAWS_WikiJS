QuizJET authoring tool provides interface to create and mofidy
[parameterized semantics
problems](http://adapt2.sis.pitt.edu/wiki/Smart_Content#Semantics_Problems_for_Java_and_Python),
organize such problems into topics, and index such problems with
underlying concepts.\n\nAn interface is shown as follows. More details
