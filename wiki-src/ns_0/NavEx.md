# Overview

\n\nNavEx - Navigation to Examples - is a Web-based adaptive navigation
tool for exploring programming examples. It serves as an adaptive shell
for accessing <a href="WebEx" class="wikilink" title="WebEx">WebEx</a>
examples. NavEx uses concept indexing of code examples and provides
three types of information to students.\n\* What examples the student is
ready to browse and what s/he is not.\n\* What is the student's current
progress.\n\* What examples are the most relevant for the student.\n\*
Progress of the student with relation to the group.\n\n= Funding =
