This tool provides the interface for teachers to define groups of
students who can access the course. It includes functionalities such as
\n\* Search/Select Group\n\* Create New Group\n\* View/Modify User
information\n\* Add/Remove User to/from Group\n\* Search User\n\*
Connect/Disconnect Group to/from Existing Applications\n\* Assign Course
to the Group\n\* Export Group Data into Common File Formats: xlsx, xls,
csv, pdf\n\* Print Group Information\n\* Notification Center\n\nAn
interface is as follows. Details can be found in the Resources
