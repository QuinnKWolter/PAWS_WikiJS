<a href="WebEx" class="wikilink" title="WebEx">WebEx</a> is a system
that serves dissected examples - <a href="#dissection" class="wikilink"
title="dissections">dissections</a>. Each dissection is a sequence of
<a href="#line" class="wikilink" title="lines">lines</a> that have
<a href="#annotation" class="wikilink"
title="annotations">annotations</a> associated with them.
<a href="#dissection" class="wikilink"
title="Dissections">Dissections</a> are grouped into collections -
<a href="#scope" class="wikilink" title="scopes">scopes</a>. The natural
domain of WebEx is programming. However, other applications are also
possible, e.g. poetry.\n\nHere are two examples from
